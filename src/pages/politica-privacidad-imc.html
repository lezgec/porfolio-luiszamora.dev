Política de Privacidad

Aplicación Calculadora IMC

Última actualización: 07 de September de 2025

1. Responsable

Esta aplicación es desarrollada por Luis Zamora (en adelante, “el desarrollador”). Para consultas de privacidad, utiliza el correo: calculadoraimc@luiszamora.dev.

2. Información que recopilamos

La aplicación no recopila, almacena ni comparte datos personales de los usuarios.

Los valores ingresados (peso y altura) se procesan solo de forma local en el dispositivo para calcular el Índice de Masa Corporal (IMC). No se envían a servidores ni a terceros.

3. Uso de la información

Los datos introducidos por el usuario se utilizan exclusivamente para mostrar el resultado del IMC en pantalla y no se conservan una vez se cierra o reinicia la aplicación.

4. Permisos

La aplicación no solicita permisos como acceso a cámara, micrófono, contactos, ubicación u otros.

5. Servicios de terceros

La aplicación no integra SDKs ni servicios de terceros que recopilen información personal (por ejemplo, analítica o publicidad).

6. Seguridad

Aunque la app no trata datos personales, se aplican buenas prácticas de desarrollo y seguridad para ofrecer un uso confiable.

7. Menores de edad

La aplicación puede ser utilizada por todo público y no está dirigida específicamente a menores. No se recopila información personal de menores de 13 años.

8. Cambios a esta política

Podremos actualizar este documento para reflejar mejoras o requisitos legales. La fecha de “Última actualización” se ajustará cuando corresponda.

9. Contacto

Si tienes preguntas sobre esta Política de Privacidad, escribe a: calculadoraimc@luiszamora.dev.
